Este repositorio contiene el código fuente de mi portafolio personal, desarrollado utilizando la biblioteca [React](https://es.react.dev/). Este proyecto forma parte de mi proceso de formación en el desarrollo Frontend con React.

## Funcionalidade

- **Presentación personal:** Muestra información sobre mí, incluyendo mi educación, habilidades, experiencia y proyectos destacados.
- **Portafolio de proyectos:** Muestra una selección de proyectos que he completado, con enlaces a los repositorios correspondientes y, si están disponibles, demostraciones en vivo.
- **Contacto:** Proporciona formas de ponerse en contacto conmigo, ya sea a través de un formulario de contacto o enlaces a mis perfiles en redes sociales.

## Tecnologías Utilizadas

- **React:** Utilizado para construir la interfaz de usuario del portafolio.
- **HTML/CSS:** Utilizado para estructurar y diseñar la apariencia del sitio web.
- **JavaScript:** Lenguaje de programación utilizado para agregar interactividad a la aplicación.
- **React Router:** Utilizado para la navegación dentro del sitio web.

## Instalación
**1.** Clona este repositorio en tu máquina local utilizando git clone.

```bash
git clone https://github.com/The-Asintota/portafolio.git
```

**2.** Navega al directorio del proyecto.

```bash
cd portafolio
```

**3.** Instala las dependencias utilizando npm o el gestor de paquete que uses.

```bash
npm install
```

**4.** Una vez que hayas clonado el repositorio y hayas instalado las dependencias, puedes ejecutar el proyecto localmente.

```bash
npm run dev
```
